# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.0   | :white_check_mark: |

## Reporting a Vulnerability (脆弱性報告について)

If you found any security vulnerabilities in fuzzuf, please report to security@ricsec.co.jp. We will discuss the response among a development team, and patch as soon as possible if we are going to respond as a bugfix.

当プロジェクトにおけるセキュリティ脆弱性を発見された方は、Eメールにて security@ricsec.co.jp までご報告ください。
ご報告をもとに、社内開発チームで脆弱性への対応を協議し、バグフィックスとして対応する場合には、可能な限り早くそのパッチを開発チームが作成し、適用します。
